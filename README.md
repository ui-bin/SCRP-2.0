## Usage
1. Download YSI @5.x
1. Download all plugins like a_mysql latest 
1. Configure SQL on gamemode (SQL_HOSTNAME, SQL_USERNAME, SQL_PASSWORD, SQL_DATABASE)
1. Compile
1. Run it on your server!
